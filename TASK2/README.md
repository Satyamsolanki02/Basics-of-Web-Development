# 🚀 ApexPlanet - Intermediate HTML, CSS & JavaScript Task

## 📌 Project Overview
This project demonstrates **modern front-end development** skills including:
- **Responsive layouts** using **Flexbox** (Navigation) & **CSS Grid** (Content areas)
- **Interactive Contact Form** with **real-time JavaScript validation** (required fields & email format)
- **Dynamic To-Do List** with full **DOM manipulation** (add, delete, toggle completion)
- **Glassmorphism UI** & fully responsive design for all screen sizes

## 🎯 Tasks Covered
| Step | Objective | Implementation |
|------|-----------|----------------|
| 1 & 2 | Contact Form + JS Validation | Form with text, email, textarea; validates required fields & email pattern |
| 3 | Responsive Layout | Flexbox navbar, CSS Grid main section + media queries |
| 4 | Dynamic To-Do List | Add/remove tasks, toggle completion, localStorage persistence |

## 🛠️ Technologies Used
- **HTML5** (Semantic structure)
- **CSS3** (Flexbox, Grid, Media Queries, Glassmorphism, Animations)
- **JavaScript (ES6)** (DOM manipulation, Event handling, Form validation, LocalStorage)
